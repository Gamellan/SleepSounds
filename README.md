# Sleep Sounds

Offline sleep sounds / white noise app for a broad global audience:
- Babies and parents
- People with insomnia
- Students
- Meditation and focus users

Included sounds (packaged inside app):
- Rain
- Ocean
- Forest
- Fan
- White Noise

## Architecture

- No backend
- No internet required
- All audio files bundled as assets

## Commercial usage notes

This repository currently includes synthetic WAV tracks generated by script:
- scripts/generate_audio.ps1

For production, you can keep synthetic tracks or replace with licensed assets.

## Run

```bash
flutter pub get
flutter run
```

## Validate

```bash
flutter analyze
flutter test
```
